## 🛠️ Technologies utilisées

Pour ce projet, j'ai privilégié la performance et la simplicité avec du code natif :

*   **HTML5** : Structure sémantique et SEO.
*   **CSS3** : Flexbox/Grid, animations personnalisées et responsive design.
*   **JavaScript (ES6+)** : Manipulation du DOM, gestion du mode sombre et animations au scroll.
*   **Icons** : [FontAwesome](https://fontawesome.com) / [Lucide Icons] / [Remix Icon].

---

## 🌟 Fonctionnalités clés

- 📱 **Full Responsive** : Adapté aux mobiles, tablettes et ordinateurs.
- 🌓 **Dark Mode** : Basculement automatique ou manuel du thème.
- ✉️ **Formulaire de contact** : Intégration avec [Formspree/EmailJS].
- ⚡ **Animations fluides** : Utilisation de la bibliothèque [AOS](https://github.io) ou CSS native.
